# SeT-MvSC

Matlab Code for multiview subspace clustering:

Reference: Zhuo Liu, Dong Hu, and Zhi Wang*. Nonconvex Low-Rank Tensor Representation with Self-Enhanced Structure Penalty for Multi-View Subspace Clustering, Submitted, 2025.

This code was authored by Zhuo Liu under the supervision of Dr. Zhi Wang at Southwest University, 2024.

For algorithm interpretation, please read our paper, in which more details are demonstrated.
